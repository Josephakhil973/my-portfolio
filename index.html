Project

JOB PORTAL SYSTEM

The Job Portal System is a web application designed to connect job seekers with employers, structured into three modules: Job Seeker, Job Provider, and Admin. The Job Seeker module allows users to create profiles, search for job openings, and apply to positions. The Job Provider module enables employers to post job vacancies and manage applications, while the Admin module oversees user management and system security. For development, I used Angular, HTML5, CSS3, and JavaScript for a responsive front end. Git supported version control, ensuring smooth code management, while Trello and Agile methodology helped organize tasks and maintain an iterative, adaptable workflow. The result is a user-friendly platform that streamlines job searches and hiring processes..
